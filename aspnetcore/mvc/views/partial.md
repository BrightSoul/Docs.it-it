### <a name="partial-view-discovery"></a><span data-ttu-id="dfbbb-138">Individuazione di visualizzazione parziale</span><span class="sxs-lookup"><span data-stu-id="dfbbb-138">Partial View Discovery</span></span>

<span data-ttu-id="dfbbb-139">Quando si fa riferimento a una visualizzazione parziale, è possibile fare riferimento alla posizione in diversi modi:</span><span class="sxs-lookup"><span data-stu-id="dfbbb-139">When referencing a partial view, you can refer to its location in several ways:</span></span>

```text
// Uses a view in current folder with this name
// If none is found, searches the Shared folder
@Html.Partial("ViewName")

// A view with this name must be in the same folder
@Html.Partial("ViewName.cshtml")

// Locate the view based on the application root
// Paths that start with "/" or "~/" refer to the application root
@Html.Partial("~/Views/Folder/ViewName.cshtml")
@Html.Partial("/Views/Folder/ViewName.cshtml")

// Locate the view using relative paths
@Html.Partial("../Account/LoginPartial.cshtml")
```

<span data-ttu-id="dfbbb-140">Nelle cartelle di visualizzazione diversa, è possibile avere diverse visualizzazioni parziali con lo stesso nome.</span><span class="sxs-lookup"><span data-stu-id="dfbbb-140">You can have different partial views with the same name in different view folders.</span></span> <span data-ttu-id="dfbbb-141">Quando si fa riferimento le viste in base al nome (senza estensione file), le visualizzazioni in ogni cartella utilizzerà la visualizzazione parziale nella stessa cartella con essi.</span><span class="sxs-lookup"><span data-stu-id="dfbbb-141">When referencing the views by name (without file extension), views in each folder will use the partial view in the same folder with them.</span></span> <span data-ttu-id="dfbbb-142">È inoltre possibile specificare una visualizzazione parziale predefinito da usare, collocandola nel *Shared* cartella.</span><span class="sxs-lookup"><span data-stu-id="dfbbb-142">You can also specify a default partial view to use, placing it in the *Shared* folder.</span></span> <span data-ttu-id="dfbbb-143">La visualizzazione parziale condivisa da utilizzare per le viste che non dispongono di una propria versione della visualizzazione parziale.</span><span class="sxs-lookup"><span data-stu-id="dfbbb-143">The shared partial view will be used by any views that don't have their own version of the partial view.</span></span> <span data-ttu-id="dfbbb-144">È possibile creare una visualizzazione parziale predefinito (in *Shared*), che viene sottoposto a override da una visualizzazione parziale con lo stesso nome nella stessa cartella di visualizzazione padre.</span><span class="sxs-lookup"><span data-stu-id="dfbbb-144">You can have a default partial view (in *Shared*), which is overridden by a partial view with the same name in the same folder as the parent view.</span></span>
